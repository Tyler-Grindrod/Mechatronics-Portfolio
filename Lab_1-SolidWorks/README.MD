# Lab 1: Gear Train and Cam-Follower Assembly

## Overview
This project involved the CAD design and kinematic analysis of a mechanical drive system. [cite_start]The objective was to calculate the required input speed for a pinion gear to drive a pulley-cam assembly at a constant **27 RPM clockwise**.

## Kinematic Analysis
The system utilizes a series of gear reductions to transform the input velocity. 

**Calculations:**
* **Total Gear Ratio:** Calculated as 0.675 based on the individual ratios of the driving, vertical, and pinion gears.
* **Input Velocity:** To achieve the target output, the input pinion gear must rotate at **40 RPM**.

## SolidWorks Implementation
* **Components:** Custom designed Base, Pinion Gear, Driving Gear, Vertical Gear, Cam, and Follower.
* **Assembly:** Utilized mechanical mates and a belt/chain feature to link the pulley system.
* **Appearances:** Applied realistic materials including polished brass for the follower and blue thick glass for the base.
